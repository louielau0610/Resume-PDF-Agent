# 商业化产品路线图 (Commercial Product Roadmap v0)

> 中文优先 | Chinese-first

## 概述

`resume_pdf_agent` 目前是一个**面向商业化的研究原型**，已完成从确定性分析管线到可选 API 层的 19+ 个里程碑。本文档说明该项目如何演进为商业可行产品。

> **当前状态**：本地确定性 Agent 阶段已完成。本路线图描述的是商业化方向，尚未实施。

---

## 阶段一：本地确定性 Agent ✅

| 能力 | 状态 |
|------|------|
| Criteria-aware 简历分析 | ✅ M1-M4 |
| 真实性检查 & Bullet 增强 | ✅ M5-M6 |
| 模板匹配 & HTML/PDF 输出 | ✅ M7-M9 |
| CLI 工作流 & 静态仪表板 | ✅ M10-M12 |
| 用户确认门控 | ✅ M14 |
| JD 合规解析 | ✅ M15 |
| 可选 LLM 改写适配器 | ✅ M16 |
| PDF Backend 诊断 | ✅ M17 |
| 视觉回归检查 | ✅ M18 |
| 可选 API 层 | ✅ M19 |

---

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| LLM 幻觉 / 不实声明 | M5 真实性检查 + M14 确认门控 + M16 输出验证 |
| 用户隐私 | 本地优先架构，不上传数据 |
| 数据安全 | 可选本地部署，无强制云服务 |
| 误导性职业声明 | 明确标注 evidence_level，禁止虚构 |
| PDF 渲染一致性 | Mock backend 确定性测试 + WeasyPrint 可选 |
| 模板 IP/版权 | 仅使用原创或授权模板 |

---

## 免责声明

本项目目前不声称是商业就绪产品。它是一个**面向商业化的研究原型**，正在进行产品化迭代。所有未来里程碑均为路线图想法，未实施。
